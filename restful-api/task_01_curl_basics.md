# 🌐 Exercices d'Introduction à `curl` et aux APIs HTTP

## 🎯 Objectif

Ce projet a pour but de comprendre et de maîtriser les bases de l’outil en ligne de commande `curl`, principalement utilisé pour interagir avec des services web RESTful.  
À la fin de cet exercice, l’étudiant(e) sera capable de :

- Installer et utiliser `curl` depuis le terminal.
- Envoyer des requêtes HTTP simples (GET, POST).
- Ajouter des en-têtes personnalisés et analyser les réponses du serveur.
- Interpréter les résultats d'une requête API (contenu, statut, en-têtes, etc.).
