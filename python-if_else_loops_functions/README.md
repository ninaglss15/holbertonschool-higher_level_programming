# Python - if/else, loops, and functions

This directory contains Python scripts that demonstrate the use of if/else statements, loops, and functions. Below is a description of each task:

## Files

### 0-positive_or_negative.py
This script generates a random number and determines whether it is positive, negative, or zero. It prints the result to the console.

### 1-last_digit.py
This script generates a random number, extracts its last digit, and determines whether the last digit is greater than 5, less than 6 but not 0, or exactly 0. It prints the result to the console.

### 2-print_alphabet.py
This script prints the alphabet in lowercase, followed by a new line. It does not include any additional characters or spaces.

### 3-print_alphabt.py
This script prints the alphabet in lowercase, excluding the letters `q` and `e`, followed by a new line.

### 4-print_hexa.py
This script prints all numbers from 0 to 98 in both decimal and hexadecimal formats.

### 5-print_comb2.py
This script prints all numbers from 0 to 99, formatted as two digits, separated by a comma and a space. The last number is followed by a new line instead of a comma.

## Usage
To execute any of the scripts, run the following command in your terminal:

```bash
./<script_name>.py
```

Replace `<script_name>` with the name of the script you want to run.

## Requirements
- Python 3.x
- Linux operating system
